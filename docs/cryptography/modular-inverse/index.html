4. General Algorithm (Compact Form)‚Äã

function extended_gcd(a, b):
    if b == 0:
        return (a, 1, 0)   # gcd, x, y
    gcd, x1, y1 = extended_gcd(b, a mod b)
    x = y1
    y = x1 - (a // b) * y1
    return (gcd, x, y)

If gcd = 1, then x is the modular inverse of a mod b.
